# 🍕 Pizza Sales Analysis Dashboard

An end-to-end Business Intelligence project that leverages **SQL** for data analysis and **Power BI** for interactive dashboard creation. This project analyzes pizza sales data to uncover customer purchasing patterns, product performance, and revenue trends to support data-driven business decisions.

## 📌 Project Overview

This project focuses on transforming raw sales data into meaningful business insights. SQL was used to clean, query, and analyze the dataset, while Power BI was used to build an interactive dashboard for visualizing key performance indicators and sales trends.

## 🛠️ Tech Stack

- Microsoft SQL Server
- SQL
- Power BI
- Power Query
- DAX

## 📊 Dashboard Features

- Total Revenue
- Average Order Value
- Total Orders
- Total Pizzas Sold
- Average Pizzas per Order
- Daily Sales Trend
- Monthly Sales Trend
- Sales by Pizza Category
- Sales by Pizza Size
- Top 5 Best Selling Pizzas
- Bottom 5 Selling Pizzas
- Interactive Filters

## 📈 Key Business Insights

- 📅 Highest order volume occurs on **Friday and Saturday evenings**.
- 📅 Peak monthly sales are observed during **July** and **January**.
- 🍕 The **Classic** category contributes the highest revenue and total orders.
- 📦 **Large-sized pizzas** generate the highest sales.
- 💰 **Thai Chicken Pizza** generates the highest revenue.
- 🛒 **Classic Deluxe Pizza** records the highest quantity sold and total orders.
